SQL

# What does? Code
1 Show all databases SHOW # DATABASES;
2 Select a table SELECT table (tablename);
3 Delete table DROP FROM TABLE;
4 Queries with constraints SELECT * FROM table_name WHERE condition;
An asterisk " * " can be used to specify that the query should return all columns of the queried tables.
